To set the identity on github 

	Run

  
		** git config --global user.email "you@example.com"
  
		** git config --global user.name "Your Name"

** GIT COMMANDS 

		** git init - initialized empty git repository 
		** git status - to check status of the project
		** git add . - to add files on the staging area
		** git commit -m "your note - your note" - to save current project status 
		** git log - to check when and who save it 
		

** Uploading project to git 

		** git clone (url) - this will clone the project to your git account 
		** git push origin master - this will send your project to git 

		…or push an existing repository from the command line

		** git remote add origin https://github.com/iLongRanger/learningLaravel.git
		** git push -u origin master
